# Quick Start

This guide walks you through your first batch rename from start to finish.

## 1. Load your files

You have three ways to load files into Filera:

- Click **Open Files** to select individual files via a file picker dialog
- Click **Open Folders** to load all files in a folder (including subfolders recursively)
- **Drag and drop** files or folders directly onto the app window

Your files appear in the left panel with their current names in the **Old Name** column. The **New Name** column starts as a copy of the old name and updates live as you add tasks.

## 2. Add a rename task

In the right panel, click **File Tasks & Effects** and pick a task from the menu. For example, add **Find & Replace**:

1. Click **File Tasks & Effects**
2. Select **Find & Replace**
3. Type a search term in the **Find** field
4. Type a replacement in the **Replace** field

Watch the **New Name** column update immediately.

## 3. Refine with filters (optional)

If you only want to rename certain files, add a filter. Click **Filters** and choose one — for example **File Type Filter** to target only `.jpg` files and skip everything else.

Filtered-out files appear dimmed in the list and won't be renamed.

## 4. Choose your output mode

In the footer of the right panel, select how you want Filera to handle the renamed files:

- **Replace files in place** — renames the original files (default)
- **Copy files to new directory** — makes renamed copies, originals are untouched
- **Move files to new directory** — moves renamed files to a chosen folder

## 5. Rename

Click **Batch Rename Files**. Done.
