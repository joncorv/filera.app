# Size Filter

The **Size Filter** includes or excludes files based on whether their size is greater than or less than a threshold you set.

## Controls

| Control | Description |
|---|---|
| **Size** | The numeric size value to compare against |
| **Unit** | The size unit: Bytes, Kilobytes, Megabytes, Gigabytes, or Terabytes |
| **Greater Than / Less Than** | The direction of the comparison |

## Modes

**Greater Than:** Only files larger than the specified size remain active.

**Less Than:** Only files smaller than the specified size remain active.

## Examples

| Size | Unit | Mode | Active files |
|---|---|---|---|
| 1 | Megabytes | Greater Than | Files larger than 1 MB |
| 1 | Megabytes | Less Than | Files smaller than 1 MB |
| 500 | Kilobytes | Less Than | Files smaller than 500 KB |
| 4 | Gigabytes | Greater Than | Files larger than 4 GB |

## Tips

- Use this to separate large files (videos, RAW photos) from thumbnails or previews
- Use **Less Than** to find and rename small placeholder files or zero-byte files
- Combine with a **File Type Filter** to target large files of a specific type (e.g., large `.mp4` files only)
