# File Type Filter

The **File Type Filter** includes or excludes files based on their file extension. You can add multiple extensions to match against.

## Controls

| Control | Description |
|---|---|
| **Extension input** | Type a file extension (e.g., `jpg`, `pdf`, `mp3`) and press Enter or click **Add** |
| **Extension chips** | The list of extensions being filtered. Click the × on any chip to remove it |
| **Inclusive / Exclusive** | Toggle between inclusive and exclusive mode |

## How to add extensions

Type an extension in the text field (without the dot — type `jpg`, not `.jpg`) and either press **Enter** or click the **Add** button. The extension appears as a removable chip. Repeat to add more.

## Modes

**Inclusive:** Only files with one of the listed extensions remain active. All other file types are excluded.

**Exclusive:** Files with the listed extensions are excluded. All other file types remain active.

## Examples

Extensions added: `jpg`, `jpeg`, `png`

| Mode | Result |
|---|---|
| Inclusive | Only image files are renamed; PDFs, docs, etc. are skipped |
| Exclusive | Image files are skipped; all other types are renamed |

## Tips

- You can mix different formats of the same type: add both `jpg` and `jpeg` to cover all JPEG files
- Use Inclusive mode when you're doing a type-specific operation (e.g., adding dates to all photos)
- Use Exclusive mode when you want to rename everything *except* a protected type (e.g., leave `.exe` files untouched)
- Extensions are matched case-insensitively: `JPG` and `jpg` both match files ending in `.JPG` or `.jpg`
